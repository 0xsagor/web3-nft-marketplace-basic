# Web3 NFT Marketplace Basic

This project helps beginners learn how NFT marketplaces work on blockchain.

You will learn:
- How NFTs are listed for sale
- How to connect a wallet
- How to buy an NFT
- How to read marketplace data
- How Web3 trading works

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-nft-marketplace-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- market.js = Marketplace contract functions  
- web3.js = Blockchain connection  
- deploy.js = Contract deploy script  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Connect wallet  
- List NFT for sale  
- Buy NFT  
- Show marketplace items  
- Simple trading flow  

## Learning Goal
This project explains how NFT trading platforms work in a simple way.
